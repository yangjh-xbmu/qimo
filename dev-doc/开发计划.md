### 2.4 详细开发步骤（5天计划）

#### 第1天：项目设置与基础组件

```bash
# 1. 创建项目
npx create-react-app personal-website --template typescript
cd personal-website

# 2. 安装依赖
npm install -D tailwindcss postcss autoprefixer
npm install lucide-react
npm install -D @types/node

# 3. 初始化Tailwind
npx tailwindcss init -p

# 4. 配置Cloudflare Pages
npm install -D wrangler
```

#### 第2天：页面结构与静态内容

```typescript
// 实现所有展示组件
// 1. Hero组件 - 个人信息展示
// 2. Works组件 - 作品网格展示
// 3. Timeline组件 - 时间线展示
// 4. Values组件 - 价值观卡片
// 5. Scholars组件 - 学者网格

// 重点：确保所有组件在移动端表现良好
```

#### 第3天：对话功能集成

```typescript
// 1. 创建ChatInterface组件
// 2. 实现消息发送/接收逻辑
// 3. 添加话题验证前端逻辑
// 4. 创建自定义hook useChat
// 5. 设计聊天界面UI

// API集成流程：
// 前端输入 → 话题验证 → 发送到Cloudflare Function → DeepSeek API → 返回结果
```

#### 第4天：样式优化与响应式

```typescript
// 1. 完成所有组件的响应式设计
// 2. 添加交互动画和过渡效果
// 3. 优化图片加载（懒加载、WebP格式）
// 4. 确保无障碍访问（ARIA标签、键盘导航）
// 5. 性能优化（代码分割、资源预加载）
```

#### 第5天：测试与部署

```bash
# 1. 功能测试
# - 对话功能测试
# - 移动端响应测试
# - 不同浏览器测试

# 2. 部署到Cloudflare Pages
# 2.1 登录Cloudflare
wrangler login

# 2.2 创建项目
wrangler pages project create personal-website

# 2.3 设置环境变量
wrangler secret put DEEPSEEK_API_KEY

# 2.4 部署
npm run build
wrangler pages deploy ./build

# 3. 配置自定义域名（可选）
```
